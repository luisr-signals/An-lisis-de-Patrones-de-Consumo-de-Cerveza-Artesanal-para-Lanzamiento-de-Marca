# Dashboard Blueprint - Monte Bravo Brewing Co.

## Enfoque

Dashboard ejecutivo en Excel para evaluar el lanzamiento de una marca de cerveza artesanal con datos recolectados en León Beer Fest 2023. La pieza funciona como herramienta de consultoría: clara, visual y orientada a decisiones de activación comercial.

## Objetivo del Dashboard

Responder de forma rápida:

- Qué estilos de cerveza tienen mayor potencial.
- Qué factores influyen más en la elección.
- Qué contextos sociales favorecen el consumo.
- Qué canales ayudan al descubrimiento de marca.
- Qué segmentos de audiencia deben priorizarse para el lanzamiento.

## Diseño Visual

- Fondo claro cálido, inspirado en papel, malta y madera clara.
- Encabezado oscuro tipo etiqueta premium.
- Acentos en ámbar, lúpulo verde, cacao y berry.
- Tarjetas limpias para KPIs.
- Barras construidas directamente en Excel para asegurar compatibilidad visual.
- Secciones compactas con lectura ejecutiva.

## Secciones del Dashboard

1. **Resumen Ejecutivo**
   - Participantes
   - Preferencia craft
   - Conocimiento alto
   - Consumo promedio válido
   - Top driver
   - Top contexto

2. **Preferencias Cerveceras**
   - Estilos favoritos
   - Estilos ancla para lanzamiento

3. **Drivers de Elección**
   - Sabor
   - Recomendación
   - Diseño del empaque
   - Precio

4. **Contexto y Canales**
   - Contextos sociales de consumo
   - Canales de descubrimiento de marca

5. **Segmentos de Activación**
   - Catadores craft
   - Socializadores
   - Exploradores de entrada
   - Buscadores de experiencia
   - Consumidor casual

## Narrativa Recomendada

El dashboard no debe sentirse como una encuesta genérica. Debe presentarse como una herramienta de inteligencia de mercado para decidir posicionamiento, experiencia de evento, comunicación y activación comercial.
