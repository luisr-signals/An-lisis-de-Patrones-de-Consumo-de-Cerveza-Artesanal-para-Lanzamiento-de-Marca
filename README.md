# An-lisis-de-Patrones-de-Consumo-de-Cerveza-Artesanal-para-Lanzamiento-de-Marca
Este proyecto de Business Intelligence analiza datos de consumo de cerveza artesanal (encuestas, ventas históricas y tendencias de redes sociales) para guiar el lanzamiento estratégico de una nueva marca. Mediante Power BI y herramientas de análisis (Python, SQL)
